# KR Mangalam University StudentFreelance Platform

**🎓 EXCLUSIVE FOR KR MANGALAM UNIVERSITY STUDENTS**

This is a specialized freelance marketplace created exclusively for KR Mangalam University students to connect, collaborate, and earn money while gaining valuable experience.

## 🎯 **University-Specific Features**

### **Student Verification System**
- ✅ **KR Mangalam Email Only** - Registration requires `@krmangalam.edu` email
- ✅ **Student ID Verification** - Valid KR Mangalam student ID required
- ✅ **Branch/Department Selection** - All KR Mangalam branches listed
- ✅ **University Confirmation** - Must confirm KR Mangalam student status

### **Available Branches**
- **Engineering**: Computer Science, IT, Electronics, Mechanical, Civil, Electrical
- **Management**: MBA, BBA, B.Com
- **Computer Applications**: BCA, MCA
- **Science**: B.Sc, M.Sc
- **Other**: All KR Mangalam departments

### **For KR Mangalam Students**
